# How to use Excel Add-Ins in the Spreadsheet Control


<p>This example illustrates how to use Excel Add-Ins in the SpreadsheetControl.<br />
The application runs MicrosoftÂ® ExcelÂ® in the background to evaluate a custom function located in the Excel Add-In macro code and displays the result in the SpreadsheetControl.
