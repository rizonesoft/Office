Developer Express Example
How to customize copy and paste commands

This example illustrates how to override CopySelectionCommand and PasteSelectionCommand behavior via the IRichEditCommandFactoryService substitution. This approach is based on the http://www.devexpress.com/scid=E2224 example. Customized CopySelectionCommand contains code for copying selection to the Clipboard (http://msdn.microsoft.com/en-us/library/system.windows.clipboard.aspx) in an HTML Clipboard Format (http://msdn.microsoft.com/en-us/library/aa767917(v=vs.85).aspx) (see the HtmlHelper.GetHtmlClipboardFormat() method), which is not implemented by default at the XtraRichEdit source code level. This allows you to copy RichEditControl content and paste it directly to a certain control that can accept HTML-formatted data from the clipboard (e. g. ASPxHtmlEditor (http://documentation.devexpress.com/#AspNet/clsDevExpressWebASPxHtmlEditorASPxHtmlEditortopic)). Note that images are embedded into resultant HTML according to the Data URI scheme (http://en.wikipedia.org/wiki/Data_URI_scheme) specification to avoid dependency on the image location (it may not be resolved by the target application). Refer to the http://www.devexpress.com/scid=E2216 example to learn more on this approach. As for customized PasteSelectionCommand, it simply retrieves plain (unformatted) text from the clipboard and pastes this text into the RichEditControl. See Also: SubDocument.GetText Method (http://documentation.devexpress.com/#CoreLibraries/DevExpressXtraRichEditAPINativeSubDocument_GetTexttopic) SubDocument.GetRtfText Method (http://documentation.devexpress.com/#CoreLibraries/DevExpressXtraRichEditAPINativeSubDocument_GetRtfTexttopic) SubDocument.GetHtmlText Method (http://documentation.devexpress.com/#CoreLibraries/DevExpressXtraRichEditAPINativeSubDocument_GetHtmlTexttopic)
